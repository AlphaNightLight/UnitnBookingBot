\doxysubsection{Detailed Description}
Functions to retrieve the values from the database. 

This file contains the implementation of the functions with reading access to the database, i.\+e. that can\textquotesingle{}t modify the records contained in it but can only retrieve values. They are grouped into four categories\+:

1) Functions to read from \char`\"{}users\char`\"{} table

2) Functions to read from \char`\"{}fairs\char`\"{} table

3) Functions to read from \char`\"{}events\char`\"{} table

4) Functions to read from \char`\"{}slots\char`\"{} table
