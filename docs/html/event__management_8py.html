Detailed Description

Handlers that manipulate the events and associated slots.

This file contains the implementation of the Telegram Bot Handlers that manage the lifecycle of events and slots associated to them. They are triggered by the following commands: /publish, /changedes, /newslot /deleteslot, /deleteevent, /myevents.
