\doxysubsection{Detailed Description}
Handlers for general information. 

This file contains the implementation of the Telegram Bot Handlers that manage the generic commands {\ttfamily /start} and {\ttfamily /help}, together with the fallbacks for invalid commands and command-\/less text.
